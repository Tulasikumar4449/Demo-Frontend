# Saheya Verification Badge System

## Badge Types & Display

### 1. **Verified Worker Badge** ✓

**Appearance:**
- Green badge with checkmark
- "✓ Verified Worker" text
- Displayed on worker profile header

**Eligibility:**
- All documents submitted and verified ✓
- Police verification complete ✓
- Background check passed ✓
- Training completed ✓
- Partnership agreement signed ✓

**Benefits:**
- Priority in job allocation
- Can charge premium rates (10-20% higher)
- Access to all job categories
- Insurance coverage active
- Eligible for Pro tier

---

### 2. **Verification In Progress Badge** 🔄

**Appearance:**
- Blue badge with refresh icon
- "🔄 Verification in Progress" text
- Shown during verification process

**Meaning:**
- Documents submitted but not fully verified
- Can browse jobs but limited access
- Cannot accept high-value jobs yet

**Next Steps:**
- Complete pending verification steps
- Upload any requested additional documents
- Wait for police clearance

---

### 3. **Pending Verification Badge** ⏳

**Appearance:**
- Amber/Yellow badge with hourglass
- "⏳ Pending Documents" text
- Indicates incomplete submission

**Action Required:**
- Upload missing documents
- Complete profile information
- Start verification process

---

### 4. **Saheya Pro Badge** ⭐

**Appearance:**
- Gold/Premium badge with star
- "⭐ Saheya Pro" text
- Highest tier recognition

**Eligibility:**
- Already verified
- 4.5+ star rating
- 100+ jobs completed
- 95%+ completion rate
- 6+ months on platform

**Premium Benefits:**
- Lowest commission rate (12%)
- First access to premium jobs
- Higher visibility in search
- Monthly bonuses
- Dedicated support manager

## Status Indicators

### Document Status Colors:

| Status | Color | Icon | Meaning |

| Verified | Green | ✓ | Document authenticated |
| Pending | Amber | ⏳ | Awaiting review |
| In Progress | Blue | 🔄 | Under verification |
| Rejected | Red | ✗ | Invalid/Needs re-upload |

### Overall Verification Status:

1. **Not Started** (Gray)
   - No documents uploaded
   - Registration incomplete

2. **In Progress** (Blue)
   - Some documents submitted
   - Verification ongoing

3. **Verified** (Green)
   - All checks complete
   - Full platform access

4. **Rejected** (Red)
   - Failed verification
   - Fraud detected
   - Policy violation

## Common Issues & Solutions

### Issue: Document Rejected
**Solutions:**
- Ensure document is clearly visible
- Check expiry date
- Verify name matches across documents
- Re-upload in better quality
- Contact support if issue persists

### Issue: Police Verification Delayed
**Solutions:**
- Check application status online
- Visit local police station if needed
- Submit additional ID proof
- Contact Saheya support for follow-up

### Issue: Training Not Completing
**Solutions:**
- Ensure stable internet connection
- Complete all modules sequentially
- Pass assessment tests
- Retake if failed (unlimited attempts)

### Issue: Badge Not Showing
**Solutions:**
- Refresh the app
- Clear cache and reload
- Check verification status page
- Contact technical support

## Trust Score Calculation

### Factors:
1. **Verification Level** (30%)
   - Document completeness
   - Background check status

2. **Performance Metrics** (40%)
   - Customer ratings
   - Job completion rate
   - Response time

3. **Experience** (20%)
   - Total jobs completed
   - Time on platform
   - Service categories

4. **Customer Feedback** (10%)
   - Review quality
   - Repeat customers
   - Complaints resolved

### Score Ranges:
- **90-100**: Excellent (Pro eligible)
- **75-89**: Very Good (Verified)
- **50-74**: Good (Partial)
- **Below 50**: Needs Improvement
